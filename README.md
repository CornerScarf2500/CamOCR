# CamOCR - Local Hybrid OCR

**Beautiful local web UI for mixed-language + handwriting OCR**  
Runs on **Windows, macOS, Linux** (x86, x64, ARM64) including Snapdragon X Plus.  
Auto-collects data for future training. Open-source Apache 2.0.

## ✨ Features
- One-click start on any OS (auto venv + install)
- Live OCR with red bounding boxes
- Contribute tab with training-data opt-out (only visible on public demo)
- Full Terms & Conditions tab
- Collected images saved locally
- Converter UI for your custom Paddle models
- Ready for Hugging Face public demo (auto-sync from GitHub)

## 🚀 Quick Start (choose your OS)

### Windows (including Surface Pro 11 Snapdragon ARM64)
1. Download or clone the repo
2. Double-click `run.bat`
3. Wait for first-time install (1-2 min) → browser opens http://127.0.0.1:7860

### macOS / Linux (including WSL)
```bash
cd CamOCR
chmod +x run.sh
